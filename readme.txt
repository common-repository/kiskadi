== Installation ==

= From your WordPress dashboard =

1. Visit _Plugins > Add New_.
1. Search for _Kiskadi_.
1. Click in _Install Now_ and, after, in _Activate_.
1. Visit _Settings > Kiskadi_ and set your API Key.

= From WordPress.org =

1. Download _Kiskadi_
1. Unzip the file and upload the _kiskadi_ directory to your _wp-content/plugins_ directory using your favorite method (ftp, sftp, scp, etcâ€¦). Or visit _Plugins > Add New > Upload Plugin_, select _ksikadi.zip_ file, click in _Install Now_ and, after, in _Activate Plugin_.
1. Visit _Settings > Kiskadi_ and set your API Key.
